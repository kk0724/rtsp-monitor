# 智能监控助手 - 使用说明

## 软件简介

智能监控助手是一款基于摄像头实时监控的智能提醒工具。它可以实时监控指定区域，当检测到有人或物体运动时，自动弹出窗口提醒，并可选择自动隐藏其他应用程序窗口。

## 主要功能

### 1. 实时监控
- 支持RTSP网络摄像头
- 可设置监控区域（ROI - 感兴趣区域）
- 实时显示FPS和运动检测状态

### 2. 智能提醒
- 检测到运动时自动弹出窗口
- 窗口闪烁红色边框吸引注意
- 播放提示音
- 系统托盘弹出通知

### 3. 窗口管理
- 可设置在检测到运动时自动隐藏指定应用程序
- 支持精确匹配和模糊匹配窗口名称
- 8秒后自动恢复被隐藏的窗口

### 4. 系统托盘
- 程序可最小化到系统托盘运行
- 托盘菜单包含所有常用功能
- 双击托盘图标显示窗口
- 检测到运动时自动从托盘弹出

### 5. 自动隐藏
- 弹出窗口后10秒自动隐藏回托盘
- 避免长时间占用屏幕空间

## 快速开始

### 1. 修改摄像头地址
在代码中找到以下行，修改为您的摄像头RTSP地址：

```python
self.camera_url = 'rtsp://admin:guest123@192.168.6.105:554/Streaming/Channels/101'
```

格式：`rtsp://用户名:密码@IP地址:端口/路径`

### 2. 调整监控区域
根据需要调整ROI区域（监控范围）：

```python
self.roi = {
    "x": 360,    # 起始X坐标
    "y": 400,    # 起始Y坐标
    "w": 1170,   # 宽度
    "h": 1040    # 高度
}
```

### 3. 运行程序

```bash
python monitor.py
```

## 详细配置说明

### 运动检测阈值
当前设置为检测画面3000像素变化触发报警（对应实际画面约12,000像素）：

```python
change_detected = main_pixels > 3000
```

如需调整灵敏度，修改这个数值：
- 数值越小越灵敏
- 数值越大越不灵敏

### 连续检测次数
需要连续几帧检测到运动才触发报警：

```python
self.min_consecutive_changes = 2  # 2帧连续变化才报警
```

### 自动隐藏时间
检测到运动后自动隐藏回托盘的时间（毫秒）：

```python
self.auto_hide_timer.start(10000)  # 10秒
```

## 使用指南

### 基本操作
1. **启动程序**：运行后会自动显示监控画面
2. **拖动窗口**：按住鼠标左键可拖动窗口到任意位置
3. **窗口置顶**：窗口始终保持在最前
4. **退出程序**：按ESC键或右键托盘图标选择"退出"

### 系统托盘功能
右键托盘图标可看到以下选项：
- **显示窗口**：显示主监控窗口
- **隐藏到托盘**：手动隐藏窗口到托盘
- **设置隐藏窗口**：配置要自动隐藏的应用程序
- **退出**：完全退出程序

### 设置隐藏窗口
1. 右键托盘图标，选择"设置隐藏窗口"
2. 输入要隐藏的窗口标题（如："无标题 - 记事本"、"微信"、"MuMu安卓设备"）
3. 点击"查看当前打开的窗口"查看所有可见窗口
4. 点击"测试查找窗口"验证是否能找到
5. 点击"保存"完成设置

### 运行效果
- **正常状态**：窗口显示监控画面，右下角显示FPS和时间
- **检测到运动**：窗口闪烁红色，播放提示音，自动弹出（如在托盘）
- **自动隐藏**：10秒后窗口自动隐藏回托盘
- **隐藏其他窗口**：如设置了隐藏窗口，检测到运动时会自动隐藏该应用

## 常见问题

### Q: 程序无法连接摄像头？
A: 检查摄像头地址是否正确，网络是否通畅，摄像头是否支持RTSP协议。

### Q: 检测不灵敏或误报过多？
A: 调整运动检测阈值：增大阈值减少误报，减小阈值提高灵敏度。

### Q: 找不到要隐藏的窗口？
A: 使用"查看当前打开的窗口"功能查看所有可见窗口的准确标题，然后复制粘贴到输入框。

### Q: 程序启动时有libpng警告？
A: 已通过环境变量屏蔽，不影响使用。

### Q: 如何完全退出程序？
A: 右键托盘图标选择"退出"，或按ESC键。
